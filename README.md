# Guess the Number

A simple number guessing game created by me, assigned as a Sleepover Project.

## 🚀 How to Play?
Just guess the number! Input any number of your guess in the textbox and check if it's correct. The website will tell you whether it's too high or too low, or maybe correct!

## 🛠️ Tech Stack
* **Structure:** HTML5
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Logic:** JavaScript
* **Other:** [tsparticles](https://github.com/tsparticles/tsparticles)
